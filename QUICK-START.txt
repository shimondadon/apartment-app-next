🛠️ TROUBLESHOOTING
────────────────────────────────────────────────────────────────

  Backend won't start?
  └─ cd cleaning-app-backend
     rm -rf node_modules
     npm install
     npm run dev

  Frontend won't start?
  └─ cd cleaning-app-frontend
     rm -rf node_modules
     npm install
     npm start

  Port already in use?
  └─ npx kill-port 3001  (backend)
     npx kill-port 4200  (frontend)

  CORS errors?
  └─ Check next.config.ts
     Ensure http://localhost:4200 is allowed

  Can't login?
  └─ Check backend console for errors
     Use valid codes: test, 1234, עובדת1234
